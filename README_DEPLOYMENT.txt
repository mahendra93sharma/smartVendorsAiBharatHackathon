================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: Login doesn't work
Solution: Try: vercel login your-email@example.com

Problem: Build fails
Solution: Test locally first: npm run build

Problem: Command not found
Solution: Run: npm install -g vercel
