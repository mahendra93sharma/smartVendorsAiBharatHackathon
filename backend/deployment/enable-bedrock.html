⚠️ Action Required: You need to enable AWS Bedrock model access to use transaction extraction features.
1 Open AWS Bedrock Console
Click the button below to open the AWS Bedrock Model Access page in your region (ap-south-1).

🔗 Open Bedrock Console
2 Request Model Access

On the Model Access page:

  • Click the "Manage model access" or "Edit" button
  • Scroll down to find Anthropic models
  • Check the boxes for Claude models:
    • ✓ Claude 3.5 Sonnet
    • ✓ Claude 3 Sonnet
    • ✓ Claude 3 Haiku
    • ✓ Claude 2.1
    • ✓ Claude 2
  • Click "Save changes" at the bottom
  • Wait for approval (usually instant)
3 Verify Access
After approval, run this command in your terminal to verify:
cd backend
python deployment/validate_prerequisites.py
You should see: ✓ AWS Bedrock: Available with Claude models
✅ Success! Once you've completed the steps above, your backend will be able to use AWS Bedrock for transaction extraction.
